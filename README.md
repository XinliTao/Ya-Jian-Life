# 雅鉴生活志

> 为用户提供商家信息查询、秒杀优惠券、智能客服等功能，同时帮助商家推广优惠信息的综合服务平台。

## 📘 项目简介

**雅鉴生活志**旨在打造一个为用户提供本地商家信息查询、优惠券领取、智能客服问答等实用服务的平台，同时为商家提供推广渠道，提高曝光度与用户转化率。

平台通过整合信息服务、智能对话与缓存系统，加速访问体验、提升用户满意度，并保障高并发场景下的数据一致性与系统稳定性。

---

## 🛠 技术栈

**后端：** SpringBoot + MySQL + Redis + Caffeine + MybatisPlus + Lua + LangChain4j
**其他：** 阿里云百炼大模型（Function Calling）、本地缓存、二级缓存架构

---

## ⭐ 核心功能与亮点

### 1. 📦 Redis缓存体系

* 使用 Redis 存储库存与订单信息，并通过 Lua 脚本保证操作的**原子性**，确保“**库存不过卖，一人一单**”。
* 更新数据库后删除缓存；若删除失败则使用队列补偿机制，确保 **TTL 兜底保证数据一致性**。
* 使用 **逻辑过期** 防止缓存击穿，根据业务场景合理设置 Redis Key 与过期机制。

### 2. ⚡ Caffeine 本地缓存 + Redis 二级缓存架构

* 提高热点数据访问速度，显著降低 Redis 压力。
* 结合滑动窗口限流，实现动态限流策略，防止系统过载、爬虫攻击、刷券、刷单等行为。

### 3. 🔐 AOP + 注解式接口保护

* 使用 AOP 与注解实现滑动窗口限流，可针对**接口、IP、用户多维度**配置限流策略。

### 4. 🤖 智能客服（含 Function Calling）

* 接入阿里云百炼大模型，实现智能客服对话。
* 支持：

  * 语义理解与上下文记忆
  * 商家信息查询、优惠券推送
  * 预约到店、服务说明等业务能力

### 5. ☁ 高可用与高并发设计

* 使用队列进行缓存删除补偿，确保最终一致性。
* 防止缓存穿透、击穿、雪崩的全链路策略。
* 部分功能使用 Lua 提升并发下的执行性能。

## 🚀 启动方式

### 1. 克隆项目

```bash
git clone https://github.com/XinliTao/Ya-Jian-Life.git
```

### 2. 配置数据库

* 导入 SQL 文件
* 在 `application.yml` 中配置数据库与 Redis

### 3. 启动项目

```bash
mvn spring-boot:run
```
